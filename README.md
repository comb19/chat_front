## 概要
チャットアプリのフロントエンドです。[バックエンド](https://github.com/comb19/chat_back)<br/>
WebSocketを使用しリアルタイムなメッセージのやり取りを実現しています。

## 使用技術
バックエンド：Go, gin<br/>
フロントエンド：TypeScript, Next.js<br/>
開発、実行環境：Docker Compose<br/>
認証：Clerk<br/>
DB：PostgreSQL<br/>
ホスティング：Render.com, Netlify.com、neon.tech<br/>
